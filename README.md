# Projeto Pessoal, Anime Fights
![Visual Studio Code](https://img.shields.io/badge/Visual%20Studio%20Code-0078d7.svg?style=for-the-badge&logo=visual-studio-code&logoColor=white)

## Descrição:
Este projeto foi uma atividade recomendada pelo professor no meu curso técnico, a ideia era basicamente escolher um tema e com este tema formar um site para si como portfolio, feito em HTML, CSS e Javascript. Eu como adoro animes e gosto da mídia japonesa, fiz um site que comparasse a animação de duas lutas dos meus animes favoritos, Dragon Ball e Jujutsu Kaisen. No site eu demonstro minha opinião sobre como a animação evoluiu ao longo dos anos, além de compartilhar momentos desses dois animes que tanto gosto.
